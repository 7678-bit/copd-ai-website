Machine Learning Overview

Machine learning (ML) is a subset of AI that allows systems to learn from data without being explicitly programmed. In healthcare, ML models can identify patterns in complex datasets to predict disease progression and outcomes.

Algorithms Used in COPD Research

  • Logistic Regression – Used for binary classification such as predicting exacerbation (yes/no).
  • Support Vector Machines (SVM) – Effective for analyzing breathing patterns and symptom prediction.
  • Random Forest – Handles large features well and used for feature importance ranking in COPD datasets.
  • Neural Networks – Ideal for processing continuous health signals and complex sensor data.

Training Data Sources

  • Hospital datasets (e.g., admissions, spirometry)
  • Wearable sensor data
  • Patient-reported symptoms and environmental data

Impact

ML models can detect subtle changes in patient data, often days before a full exacerbation occurs. These predictive models enable earlier intervention and better resource allocation.
